LB2D - Laexian_Box2D_Editor
====================

Box2D Editor Eclipse RCP based

Build/Install/Run
====================

Clone the repository.
Set up, Eclipse.
In Eclipse, Right click com.laex.cg2d.core project, Run As Eclipse Application.

Plenty of bugs.
